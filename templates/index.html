NetWatch Dashboard

  • Target guide: This Host checks only the machine running the dashboard. Single IP is best for a router, printer, NAS, or one workstation. Local Subnet is best for a full LAN sweep and will take the longest.
  • How the scan works: This build runs service detection against the selected ports, then adds lightweight enrichment with reverse DNS, MAC vendor lookup, HTTP title checks on 80/443, and simple device-type heuristics to make host labels more useful.
  • Using the actions: Start New Scan launches a new scan for the current target. Refresh / Clear Results wipes the dashboard back to an empty state. Export CSV downloads the current devices and findings for review.
